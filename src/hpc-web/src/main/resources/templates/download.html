Download

You may download a data file to your local computer (synchronous) or to a Globus endpoint location (asynchronous).
If you use asynchronous option, you will first give WRITE access of your Globus target endpoint location to the HPC DME Service account; ; Copy UUID value from your Shared Endpoint of Globus Manage Endpoints into text entry "Globus Endpoint Name (UUID)"; lastly, enter a file name into text entry "Globus Endpoint (Destination) Path" for the selected data file.
If you use synchronous option, you may leave the existing file name as is or enter a new name as the "Download file name".

You will download the selected collection of data files to a Globus endpoint location asynchronously.
Please first grant WRITE access of your Globus target endpoint location to the HPC DME Service account; Copy UUID value from your Shared Endpoint of Globus Manage Endpoints into text entry "Globus Endpoint Name (UUID)"; lastly, enter a directory name into text entry "Globus Endpoint (Destination) Path" for the selected collection.
The new directory will be created under your shared endpoint at Globus.
