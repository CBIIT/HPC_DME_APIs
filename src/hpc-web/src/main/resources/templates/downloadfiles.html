This page allows you to download the following selected collection of data files to a Globus endpoint location asynchronously.

First, grant WRITE access of your Globus target endpoint location to the HPC DME Globus Group (Refer to User Guide, Preparing to Use Globus with DME); Copy UUID value from your Shared Endpoint of Globus Manage Endpoints into text entry "Globus Endpoint Name (UUID)"; lastly, enter a directory name into text entry "Globus Endpoint (Destination) Path" for the selected collection.
The system copies the data into the directory under your shared endpoint at Globus.

This page allows you to download the following selected collection of data files to an S3 bucket asynchronously.

First, create a bucket, create an access policy, and obtain the access key with the proper permissions set up for the bucket (Refer to User Guide, Preparing to use Amazon Web Services S3 with DME); Enter the name of the destination S3 bucket, path and the name of the folder to be created in the destination bucket; Copy access key and secret key into text entry "Access Key" and "Secret Access Key"; lastly, enter the Region into text entry "Region".
The system copies the data into the directory under your S3 bucket in the specified path.
